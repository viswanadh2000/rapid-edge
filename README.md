# RAPID-Edge 🚀

RAPID-Edge is a reliability-aware, privacy-constrained adaptive inference framework for edge AI systems.

## 🔥 Key Features
- Multi-budget optimization (Reliability, Privacy, Energy, Utility)
- Adaptive inference decision engine
- Benchmark for edge AI evaluation (REAP)

## 📄 Paper
See `/paper/RAPID_Edge_Paper.pdf`

## ▶️ Run Benchmark

```bash
python benchmark/run.py
```

## 🧠 Architecture
- Context Monitor
- Budget Estimator
- Decision Engine
- Execution Modes (Local, Cloud, Split, Degraded, Abstain)

## 📊 Results
- 32% SLA improvement
- 27% privacy risk reduction
- 18% energy savings
